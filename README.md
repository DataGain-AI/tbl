# T Balaji Logistics - Modern Corporate Website

A premium, modern 4-page website for **T Balaji Logistics** built with responsive layouts, cyber-logistics color themes, interactive widgets, and smooth micro-animations.

## 📂 Project Structure

All files are located in the [tbalajilogistics](file:///D:/data-gain/tbalajilogistics) folder:
*   [index.html](file:///D:/data-gain/tbalajilogistics/index.html) - Landing page containing the live shipment tracking simulator.
*   [services.html](file:///D:/data-gain/tbalajilogistics/services.html) - Core logistics verticals and the interactive shipping rate calculator.
*   [about.html](file:///D:/data-gain/tbalajilogistics/about.html) - Company history timeline, mission values, and networks.
*   [contact.html](file:///D:/data-gain/tbalajilogistics/contact.html) - Global hub map, corporate office addresses, and inquiry form.
*   [styles.css](file:///D:/data-gain/tbalajilogistics/styles.css) - Global styling, typography, variables, responsive breakpoints, and animations.
*   [main.js](file:///D:/data-gain/tbalajilogistics/main.js) - App logic (tracking mock DB, cost estimations, and menu controls).

## 🚀 How to View Locally

You can open and view the website using any of the following methods:

### Option 1: Direct Browser Launch
Simply double-click the [index.html](file:///D:/data-gain/tbalajilogistics/index.html) file to open it directly in Google Chrome, Microsoft Edge, Safari, or Firefox.

### Option 2: Live Server (VS Code Extension)
1. Open the [tbalajilogistics](file:///D:/data-gain/tbalajilogistics) folder in VS Code.
2. Click **Go Live** on the bottom status bar (requires the "Live Server" extension).

### Option 3: Command Line Dev Servers

**Python 3:**
Run the following command in your terminal inside the directory:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your web browser.

**Node.js (npx):**
Run the following command in your terminal inside the directory:
```bash
npx serve
```
Then visit the local URL provided by the tool (usually `http://localhost:3000`).
